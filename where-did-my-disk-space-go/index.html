Where Did My Disk Space Go?

A focused macOS utility that helps you find large files, old downloads, developer caches, build artifacts, and duplicates without sending scan results off your Mac.

Built for macOS 14 or later. Cleanup uses the Trash so removed items remain recoverable until you empty it.

Find the storage you can actually act on.

Developer cleanup targets

Review Xcode DerivedData, archives, simulator data, SwiftPM caches, node_modules folders, and common package-manager caches.

Local file review

Spot older downloads and large media files in familiar folders, then reveal items in Finder before deciding what to do.

Recoverable cleanup

Selected items are moved to Trash. The app does not permanently erase files as part of its normal cleanup workflow.

Private by design.
